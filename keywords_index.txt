[A] 核心词条 GENERAL_KEYWORDS (35 个)
    已在 constants.py 中定义，引擎已实现处理逻辑。

  攻击类：
    - 迅捷        部署回合即可攻击
    - 串击        攻击指向本列所有敌方异象
    - 穿刺        攻击同时指向对手
    - 穿透        攻击穿透目标击中后方
    - 空袭        无法选中非防空异象
    - 横扫 X      同时攻击相邻X列
    - 高频 X      结算阶段可攻击X次
    - 先攻 X      同列结算优先级
    - 三重打击    攻击三次
    - 破甲 X      无视X点坚韧

  防御/生存类：
    - 坚韧 X      受到的伤害-X
    - 尖刺 X      受战斗伤害时反弹X点
    - 绝缘        不受敌方策略影响
    - 潜行        攻击后无法被选中
    - 防空        优先被空袭/视野选中，敌方失去串击/穿刺
    - 重甲 X      额外护甲
    - 冰冻 X      无法攻击X回合
    - 眩晕 X      无法行动X回合
    - 脆弱        受到额外伤害

  部署/位置类：
    - 协同        允许同列部署
    - 独行        无法与友方同列
    - 藤蔓        必须部署在友方异象上
    - 水生        只能部署在水路
    - 两栖        可以部署在水路
    - 高地        与地形相关
    - 漂浮物      部署相关

  效果触发类：
    - 部署        部署时触发效果
    - 亡语        被消灭时触发效果
    - 献祭 X      可被献祭X次
    - 丰饶 X      被献祭时提供X点鲜血
    - 回响 X      生成低费复制/返回手牌
    - 恐惧        可被清除的通用词条标记
    - 成长 X      X回合后转换形态
    - 休眠 X      试图攻击时改为减1
    - 视野 X      手动指向，攻击范围+X列
